git init -> git 초기화

git add . -> 모든 파일을 추가

git branch branch_name -> branch 생성

git pull = fetch + merge -> master에 remote repository 반영하기

git branch -d 브랜치명 -> local 브랜치 삭제

git push origin --delete 브랜치명 -> remote 브랜치 삭제

git branch -a -> local 브랜치 모두 보기

git commit -m "first commit" -> 히스토리 만들기

git remote add origin https://github.com/jungwho/github-practice.git -> 내 repository와 연결고리

git push origin master
